   427→## Unresolved Issues & Risks
   428→
   429→1. **Player data quality**: Some player names in the seed may not be perfectly accurate (the seed uses generated names for less famous players). Needs review and correction from real historical data sources (Transfermarkt, soFIFA).
   430→
   431→2. **Performance**: The spin API loads ALL club-seasons with players in one query (5000+ records). Should add caching or optimized queries for production.
   432→
