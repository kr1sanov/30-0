# История изменений 30-0 RPL

## [2025-07-05] — Исправления синхронизации, UI/UX, мобильная адаптация

### 🔧 Исправления синхронизации прогресса
- **Привязка GameRun к User**: API `POST /api/runs` теперь принимает `userId` и связывает игру с пользователем в базе данных
- **Новый API `/api/runs/active`**: GET-эндпоинт для загрузки текущего незавершённого драфта пользователя. Позволяет синхронизировать прогресс между устройствами
- **Автозагрузка активного драфта**: После авторизации через Telegram автоматически загружается последний незавершённый драфт с сервера
- **Синхронизация профиля**: После авторизации автоматически загружаются профильные данные с сервера (`loadProfileFromCloud`, `loadActiveRun`)
- **authUserId в gameStore**: Добавлено поле `authUserId` для связи с сервером, сохраняется в localStorage

### 🔧 Исправление бага "0 прокруток и пустой состав"
- **Оптимистичное обновление состояния**: `startRun()` теперь устанавливает слоты и `rerollsLeft` немедленно до завершения API-запроса
- **Обработка ошибок API**: Если `startRun()` не удаётся, пользователь возвращается на экран настройки вместо зависания на пустом драфте
- **Передача userId**: При создании игры передаётся `userId` для привязки к аккаунту

### 📱 Мобильная адаптация (Telegram Mini App)
- **Убран нижний тулбар (Footer)**: Компонент Footer полностью удалён из интерфейса
- **Новый статичный хедер**: На всех страницах отображается хедер с кнопками:
  - Слева: **Домой** (иконка дома + текст)
  - По центру: **30-0 RPL** (название приложения)
  - Справа: **Профиль** (иконка пользователя + текст)
- **Убрана поддержка пробела**: Удалён слушатель клавиатуры Space для спина (не нужен на мобильном)
- **Убран текст "или нажмите Пробел"**: Удалена подсказка с клавиатурным сокращением
- **Уменьшены отступы**: Оптимизированы паддинги для мобильного экрана

### 🏟️ Уллинение футбольного поля
- **Увеличена высота поля**: `paddingBottom` изменён с `68%` на `130%` для более вытянутого поля
- **Уменьшены карточки игроков**: Размер карточек уменьшен с 34/32px до 30/28px для предотвращения перекрытия
- **Рейтинг в карточке**: На заполненной позиции отображается рейтинг игрока вместо аббревиатуры позиции
- **Фамилия под карточкой**: Мелкий шрифт (6px) для фамилии и флага гражданства
- **Обновлён SVG viewBox**: Координаты линий связи адаптированы под новое соотношение сторон
